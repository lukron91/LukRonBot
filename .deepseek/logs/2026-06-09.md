## Sesja 2 — Fix systemu motywu (opacity, borders, modal)

**Model:** DeepSeek V3 (GitHub Copilot)

### Zadanie: Naprawa systemu motywu — opacity, accent borders, tabOpacity dla sekcji, modal

### Problem

1. **Opacity odwrócone:** CSS `rgba(r, g, b, 0.2)` = 20% widoczności, ale użytkownik ustawiał suwak na 0.2 oczekując 20% przezroczystości (80% widoczne). Kod robił `1 - transparency` co dawało odwrotny efekt.
2. **`--surface-opacity` fixed:** Wszystkie karty sekcji (`section-card`, `stat-card`, itd.) używały `--surface-opacity` zamiast `--tab-opacity`, więc suwak tabOpacity nic nie robił dla kart.
3. **Szare border zamiast accent:** `.section-card` i `.theme-section` miały `--border-color` (szary) zamiast `--accent-color`.
4. **Stare wartości w localStorage:** Po zmianie logiki opacity, stare wartości z localStorage nadal używały odwróconej logiki.

### Co zrobiono

#### 1. Fix opacity — usunięcie inwersji

**Plik:** `panel/lib/theme-context.jsx`

- Usunięto `1 - transparency` przy zapisie CSS variables
- Teraz bezpośrednie wartości: panelOpacity=0.85, tabOpacity=0.85, windowOpacity=0.92
- `--surface-opacity` fixed na 0.92 (tylko sidebar)
- Zmieniono etykiety z "Przezroczystość paneli" na "Krycie paneli: 85%"

**Przed:**
```js
const panelOpacity = 1 - (parseFloat(savedTheme.panelOpacity || '0.15'));
```

**Po:**
```js
const panelOpacity = parseFloat(savedTheme.panelOpacity || '0.85');
```

#### 2. Fix tabOpacity dla sekcji

**Plik:** `panel/app/globals.css`

Wszystkie karty sekcji zmienione z `--surface-opacity` na `--tab-opacity`:
- `.section-card` → `rgba(var(--surface-rgb), var(--tab-opacity))`
- `.stat-card` → `rgba(var(--surface-rgb), var(--tab-opacity))`
- `.settings-card` → `rgba(var(--surface-rgb), var(--tab-opacity))`
- `.health-card` → `rgba(var(--surface-rgb), var(--tab-opacity))`
- `.bot-missing-card` → `rgba(var(--surface-rgb), var(--tab-opacity))`

**Plik:** `panel/app/dashboard/moderation/users/page.jsx`
- `.user-card` → `rgba(var(--surface-rgb), var(--tab-opacity))`

**Plik:** `panel/app/dashboard/page.jsx`
- `.trend-section` → `rgba(var(--surface-rgb), var(--tab-opacity))`
- `.card-list` → `rgba(var(--surface-rgb), var(--tab-opacity))`

**Plik:** `panel/app/dashboard/roles/page.jsx`
- `.rp-section` → `rgba(var(--surface-rgb), var(--tab-opacity))`

**Plik:** `panel/app/dashboard/config/page.jsx`
- `.section` → `rgba(var(--surface-rgb), var(--tab-opacity))`

**Plik:** `panel/app/dashboard/bot-settings/page.jsx`
- `.section` → `rgba(var(--surface-rgb), var(--tab-opacity))`

#### 3. Fix accent borders

**Plik:** `panel/app/globals.css`
- `.section-card` border: `var(--border-color)` → `var(--accent-color)`

**Plik:** `panel/app/dashboard/theme/page.jsx`
- `.theme-section` border: `var(--border-color)` → `var(--accent-color)`

**Plik:** `panel/components/Modal.jsx`
